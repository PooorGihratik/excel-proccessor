ИНСТРУКЦИЯ К ИСПОЛЬЗОВАНИЮ

________________________________________________________________________________
Содержимое папки:


Основные файлы:
- main.py - файл запуска программы
- Настройки.txt - файл, в которой хранятся настройки программы (более подробная информация внутри файла)
- Files - папка по умолчанию, в которой программа будет собирать текстовые файлы (можно поменять в настройках)
Также в папке находятся 2 bat-файла, с помощью которых можно переключаться между версиями программы
(подписаны соответствующе)


Служебные файлы (не трогать):
- excel_processor - папка, содержащая остальной код, который используется программной
- .gitignore - служебный файл (используется Git)
- requirements.txt - служебный файл (содержит список библиотек, используемых программой)

________________________________________________________________________________
Использование:

Для использования необходимо установить python 3 версии и pip
(есть возможность обойтись и без этого, но много будет танцев с бубном для осуществления)

Запуск осуществляется через файл main.py
Программа будет осуществлять поиск книги с лицевыми счетами в директории программы
Имя книги можно указать в файле настроек

После обработки она создаст книгу с расшифровкой банка в директории программы
Имя можно так же указать в файле настроек
